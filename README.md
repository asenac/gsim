gsim
====

Framework for developing message-based test application.

Installation
------------

    mkdir build
    cd build
    cmake -DCMAKE_INSTALL_PREFIX=<path to install gsim> \
        -DBOOST_ROOT=<boost libraries path> ..
    make && make install
